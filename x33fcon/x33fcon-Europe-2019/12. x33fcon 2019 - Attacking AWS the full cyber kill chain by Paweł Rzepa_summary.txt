In this presentation, the speaker discusses the importance of testing cloud security and demonstrates a hypothetical attack on an AWS environment. They highlight the gap in security knowledge among professionals when it comes to the cloud and provide examples of real-world cloud security breaches. The speaker then guides the audience through an interactive cloud attack scenario, making use of tools like Paku, an AWS exploitation framework. They showcase how to exploit vulnerabilities, gain access to instance profiles, and escalate privileges. The presentation also covers techniques for hiding fingerprints, persisting access, and evading detection. The speaker concludes by offering methods for conducting a comprehensive cloud security assessment and emphasizes the responsibility of customers in ensuring cloud security.
